---
description: 翌週の目標を作成
---

来週分の `goals/YYYY-Www.md` を作成する。

## 手順

1. 直前の週の `goals/YYYY-Www.md`（最新のもの）をベースラインとして読み込む。
2. 来週の ISO週番号と日付範囲（Mon – Sun）を計算する。
3. カテゴリごとに **変更点だけ** 短く質問する:
   - "Daily Must は変更なし？"
   - "Weekly Must の OSS 目標は今週どうする？（先週: X）"
   - "Bluesky 英語投稿の目標は？（先週: X）"
   - "Bizmates の本数は？（先週: X）"
   - "Option B で挑戦したいものは？"
4. `templates/weekly.md` の構造で新ファイルを書き出す。
5. ファイルを1度表示する。
6. 作成した `goals/YYYY-Www.md` だけを `git add` し、`git commit -m "plan: YYYY-Www"` でコミット、`git push origin main` する。失敗したらユーザーに報告して終了。

## 禁止

- 目標を勝手に提案しない（聞かれたら答える）。
- 励まし・コメントを書かない。
- 過去の Must 未達を理由に目標を下げる提案はしない（評価は `/review` の役割であり、目標は本人が決める）。
- 「無理しないで」「頑張りましょう」系の表現は出さない。

## 注意

- ISO週番号は厳密に。月曜始まり、日曜終わり。
- 年をまたぐ場合（W52/W53/W01）は特に注意。
