今日の日付の日次ログを作成または開く。

## 手順

1. 今週の `goals/YYYY-Www.md`（ISO週番号）を読み、Daily Must を1行で表示する（リマインダー）。
2. 今日のログファイル `logs/YYYY/MM/YYYY-MM-DD.md` を確認:
   - 存在しなければ `templates/daily.md` から作成
   - すでにあれば追記モードで開く
3. テンプレートのフィールドを上から順に短く質問して埋める:
   - 1項目1質問
   - 「なし」「やらなかった」もそのまま記録
   - すでに埋まっているフィールドはスキップ
   - チェックボックスは [ ] → [x] に書き換え
4. 完了したらファイル全体を1度表示する。
5. 今日のログファイルだけを `git add` し、`git commit -m "log: YYYY-MM-DD"` でコミット、`git push origin main` する。失敗したらユーザーに報告して終了。

## 禁止

- コメント・励まし・評価を書かない。
- 解釈を加えない（「お疲れ様」「いい1日でしたね」など一切なし）。
- フィールドを勝手に追加しない。
- ユーザーの回答を要約・言い換えしない。原文のまま記録する。

記録は素材。解釈は週次レビュー（`/review`）で行う。
